# 🏛️ Sistema Inteligente Municipal - Municipalidad de Llanquihue

Sistema desarrollado en Python utilizando Inteligencia Artificial mediante OpenAI y LangChain, enfocado en mejorar la atención ciudadana y facilitar consultas municipales mediante un asistente virtual inteligente.

---

# 📌 Objetivo del Proyecto

La problemática principal que busca resolver este proyecto es mejorar el acceso a información municipal de manera rápida, clara y automatizada.

Actualmente muchas páginas municipales presentan problemas como:

- Navegación compleja
- Exceso de información
- Lentitud para encontrar trámites
- Falta de asistencia inmediata
- Baja automatización de atención ciudadana

Este sistema busca solucionar aquello mediante un ChatBOT inteligente capaz de:

✅ Responder consultas municipales  
✅ Mantener conversaciones contextuales  
✅ Recordar historial reciente  
✅ Detectar solicitudes frecuentes  
✅ Utilizar herramientas inteligentes  
✅ Adaptar el tipo de respuesta  
✅ Enviar correos electrónicos reales  
✅ Entregar atención automatizada
